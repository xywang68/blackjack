# blackjack

A pet project to exercise pytest and pytest-cov

#### python venv

```shell \tiny
(shell)$ source python-venv/bin/activate
(venv)$ python -m pip install -r requirements.txt
```

#### pytest

```shell \tiny
(venv)$ pytest --verbose

```

#### pytest-cov

```shell \tiny
(venv)$ pytest --cov blackjack --cov-report=term-missing

```
